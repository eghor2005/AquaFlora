# AquaFlora
AquaFlora - Arduino watering controller with TFT display.  Set watering time (1-600 sec) via potentiometer, press button to start. Dual relay outputs, visual progress bar and countdown on ST7735 display.  Hardware: Arduino, ST7735, potentiometer, button, 2 relays.  Pins: CS10, RST9, DC8 | Relay D5/D6 | Button D2 | Pot A0
